PKCS#11 is the universal, standard language that everyday software uses to talk directly to secure, physically hardened hardware devices (like smartcards, security tokens, or massive hardware security modules).

For decades, this standard language only needed to support traditional, relatively simple mathematical equations. However, to ensure that future software can securely ask these hardware devices to perform the massively complex new Post-Quantum calculations, the entire PKCS#11 vocabulary is currently being heavily rewritten and updated on a global scale.
