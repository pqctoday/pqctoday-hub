# Operating System & Platform Crypto PQC

Module covering OS-level PQC migration including system TLS policies, SSH host keys, package signing, and FIPS mode compatibility.

## Key Topics

- Linux CryptoAPI and Windows CNG PQC migration
- System-wide TLS policy: OpenSSL, GnuTLS, SCHANNEL
- SSH host key migration to ML-DSA-65
- Package signing (RPM/DEB) with ML-DSA
- FIPS mode compatibility with PQC algorithms

## Workshop Steps

1. OS Crypto Inventory
2. System TLS Configurator
3. SSH Host Key Migrator
4. Package Signing Migrator
5. FIPS Compatibility Checker
